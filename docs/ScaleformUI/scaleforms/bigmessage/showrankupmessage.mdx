# ShowRankupMessage

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>

<TabItem value="lua" label="Lua">

```lua
ScaleformUI.Scaleforms.BigMessageInstance:ShowRankupMessage("Hey this is a example", "This is a subtitle", 50, 5000, false)
```
- **Description**: Displays a rank-up message on the screen showcasing the specified title, subtitle, rank number, and time. If manualDispose is set to `true`, the duration specified will be ignored, and the displayed message will persist on the screen until Dispose is called. By default, manualDispose is set to false.
- **Parameters**:
  - `message`: The main message to be displayed.
  - `description`: The subtitle or description to accompany the main message.
  - `rank`: The rank number to be displayed.
  - `time`: The time duration for which the message is displayed, in milliseconds. Ignored if manualDispose is true.
  - `manualDispose`: (Optional) If set to `true`, the scaleform message remains on the screen until manually disposed and time is ignored.
- **Return**: None

</TabItem>
