# ShowOldMessage

import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

<Tabs>

<TabItem value="lua" label="Lua">

```lua
ScaleformUI.Scaleforms.BigMessageInstance:ShowOldMessage("Hey this is a example", 5000, false)
```
- **Description**: Displays an old-style message on the screen with the specified text and duration. If manualDispose is set to `true`, the time specified will be ignored, and the displayed message will persist on the screen until [Dispose](/scaleformui/docs/ScaleformUI/scaleforms/Dispose) is called. By default, manualDispose is set to `false`.
- **Parameters**:
  - `msg`: The main message to be displayed.
  - `time`: The time duration for which the message is displayed, in milliseconds. Ignored if manualDispose is `true`.
  - `manualDispose`: (Optional) If set to `true`, the scaleform message remains on the screen until manually disposed and time is ignored.
- **Return**: None

</TabItem>
