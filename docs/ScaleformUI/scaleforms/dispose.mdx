# Dispose


import Tabs from '@theme/Tabs';
import TabItem from '@theme/TabItem';

### Description

Unload the specified Scaleform instance from memory, ensuring its visual elements are no longer displayed on the screen.

<Tabs>
<TabItem value="lua" label="Lua">

### Method Available On Scaleforms

- `ScaleformUI.Scaleforms.BigMessageInstance`
- `ScaleformUI.Scaleforms.MidMessageInstance`
- `ScaleformUI.Scaleforms.Warning`
- `ScaleformUI.Scaleforms.Countdown`

```lua
ScaleformUI.Scaleforms.BigMessageInstance:Dispose()
```
</TabItem>

<TabItem value="csharp" label="C#">

### Method Available On Scaleforms

- `ScaleformUI.Main.BigMessageInstance`
- `ScaleformUI.Main.MedMessageInstance`
- `ScaleformUI.Main.Warning`
- `ScaleformUI.Main.Countdown`

```csharp
ScaleformUI.Main.BigMessageInstance.Dispose();
```
</TabItem>
</Tabs>
